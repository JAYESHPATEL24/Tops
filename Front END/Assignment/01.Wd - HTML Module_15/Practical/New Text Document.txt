1. HTML Basics

o A header (<header>), footer (<footer>), main section (<main>), and aside section 
(<aside>).

o A list (both ordered and unordered). 
o A link that opens in a new tab.


2.HTML Forms 
Task: Create a contact form with the following fields: 
o Full name (text input) 
o Email (email input) 
o Phone number (tel input) 
o Subject (dropdown menu) 
o Message (textarea) 
o Submit button 
 
Additional Requirements: 
 
o Use appropriate form validation using required, minlength, maxlength, and pattern. 
o Link form labels with their corresponding inputs using the for attribute.
